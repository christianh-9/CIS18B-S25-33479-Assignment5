# Notification System

Demonstrates using design patterns in Java — including **Observer**, **Abstract Factory**, **Builder**, and **Generics**  to simulate a notification system for a marketplace platform,  *MarketBridge*.

## How to run
1. git clone https://github.com/christianh-9/CIS18B-S25-33479-Assignment5.git
2. cd CIS18B-S25-33479-Assignment5
3. javac src/*.java
4. java -cp src Main
